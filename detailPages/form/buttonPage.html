框架自带颜色、大小、样式控制

通过在class属性中添加内置类名生成不同样式按钮

●a或button标签 添加 "button" 类生成按钮基本样式

●颜色控制：添加 button-positive、button-dark等类

●大小：button-small、默认、button-large

●样式：button-block、button-full、button-outline button-clear

图标按钮

查看代码 <button class="button icon-left ion-chevron-left button-dark"> 返回 </button> <button class="button icon-left ion-home"> 主页 </button> <button class="button icon-right ion-chevron-right button-dark"> 更多 </button> <button class="button button-clear button-icon icon ion-gear-a"> </button>

按钮组

查看代码 <div class="button-bar royal-bg"> <button class="button button-clear icon ion-ios-loop"> </button> <button class="button button-clear icon ion-ios-skipbackward"> </button> <button class="button button-clear icon ion-ios-skipforward"> </button> <button class="button button-clear icon ion-ios-pause"> </button> <button class="button button-clear icon ion-ios-volume-high"> </button> </div>
