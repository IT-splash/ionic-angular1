上拉菜单$ionicActionSheet

上拉菜单是自屏幕底部向上滑出的菜单

使用一个JSON对象定义上拉菜单选项

1）titleText 设置标题

2）buttons 自定义按钮组

3）cancelText 取消按钮描述

4）destructiveText 危险按钮描述，字体默认为红色

选项点击事件

1）buttonClicked 按钮点击事件

2）cancel 取消按钮点击回调

3）destructiveButtonClicked 危险按钮点击回调

4）cancelOnStateChange 切换视图是是否关闭菜单，值为 true | false，默认true

查看代码 JS: $ionicActionSheet.show({ titleText: "$ionicActionSheet", buttons: getItems(), cancelText: "取消", buttonClicked: function(index){ alert(this.buttons[index]); }, cancel: function(){ ... } })

弹出框$ionicPopup

弹出框通常用于提醒、警告

点击查看效果
查看代码 JS： //提示、确认框 $ionicPopup.confirm({ title: "确认框", template: "确定要学Ionic框架吗", okText: "必须的", cancelText: "算了吧" }).then(function(result){ if(result){ ionicLoadingShowInfo("加油哦 Ionic ！"); }else{ ionicLoadingShowInfo("学点别的也好！") } }); //警告框 $ionicPopup.alert({ title: "警告", template: "越是憧憬 越要风雨兼程", okText: "好" })

浮动框$ionicPopover

以非侵入的方式提供当前视图的额外信息

实现方式与 模态框相同

1）声明对话框模板 使用ion-popover-view指令声明对话框模板

2）使用$ionicPopover服务创建模板对象

3）使用创建的模板对象

查看代码 JS: //创建ionicPopover对象 $ionicPopover.fromTemplateUrl("ionicPopover.html",{ scope: $scope }).then(function(popover){ $scope.popover = popover; }); //显示 $scope.showIonicPopover = function($event){ $scope.popover.show($event); };
